# Tower Defense

Game of the Amazons in C.

## Authors

- Mathias APARICIO
- Hugo BASTIEN
- Mathias ROSA
- Mohamed REKIK

## Requirements

- Make
- gcc

## Compile

```bash
make build
```

## Run

```bash
make install
install/server <player1> <player2> 
```

or
    
```bash
make
make run
```
